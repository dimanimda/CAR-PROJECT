# RUBRIC v1 — оценка прогонов CAR-PROJECT

Контрольная шкала. **Заморожена** — меняем критерии только переходом на `v2` (и помечаем в `RESULTS.md`, по какой версии оценён каждый прогон, иначе прогоны несравнимы).

Оценка одного `index.html` идёт по трём блокам. Один прогон = одна заполненная строка в `RESULTS.md`.

---

## A. Автоматическая часть (binary / число — проверяю кодом и консолью)

| # | Критерий | Тип | Как проверяю |
|---|---|---|---|
| A1 | Один файл, ноль внешних библиотек | 0/1 | grep на `src=`, `import`, CDN-ссылки |
| A2 | Ноль ошибок/варнингов в console | 0/1 | открыть в браузере, прочитать console |
| A3 | Canvas на весь вьюпорт + ресайз | 0/1 | чтение кода + ресайз окна |
| A4 | Число parallax-слоёв с разной скоростью | число | подсчёт из кода (цель ≥3, хорошо ≥4) |
| A5 | Бесшовный луп (нет рывка/щели на стыке) | 0/1 | визуально на стыке + чтение wrap-логики |
| A6 | Рендер через `requestAnimationFrame` (не `setInterval`) | 0/1 | grep |
| A7 | Frame-rate-независимая анимация (delta-time) | 0/1 | чтение кода |

`auto_score` = A1+A2+A3+A5+A6+A7 (6 бинарных) + min(A4,4). Максимум **10**.

## B. Визуальная часть (шкала 1–5 каждая, две независимые колонки: мой балл / балл Димы)

| # | Критерий |
|---|---|
| B1 | Реализм машины в side-view, на переднем плане |
| B2 | Правдоподобное вращение колёс (синхронно со скроллом) |
| B3 | Subtle body motion — связь с дорогой, без «прыжков» |
| B4 | Когезивность неба и освещения |
| B5 | Общее ощущение: calm / immersive / cinematic |

`visual_score` = сумма B1..B5 (макс **25**), отдельно для меня и для Димы. Расхождение колонок — отдельный сигнал.

## C. Процессные метрики (контекст, не входят в score)

- размер кода (строк / КБ)
- время генерации
- число итераций до рабочего результата
- (если доступно) токены / стоимость

---

## Итог
`total = auto_score (0..10) + visual_score (0..25)` → макс **35**, считается отдельно по моей и по Диминой визуальной колонке.
Сравниваем ячейки матрицы между собой; абсолютное число вторично, важна разница по осям (обвязка / модель / подход).
